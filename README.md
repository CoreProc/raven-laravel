# Raven for Laravel

A Laravel 5.1 library that easily integrates raven-php to centralize your logs using Sentry.

## Install

Via Composer

``` bash
$ composer require kryo/raven-laravel
```

## Usage

``` php
$skeleton = new Coreproc\RavenLaravel();
echo $skeleton->echoPhrase('Hello, League!');
```
